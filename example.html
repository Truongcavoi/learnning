Microsoft 365導入サービス

Microsoft 365の導入を希望されるお客様のご希望の内容、予算に合わせて最適な構成を行えるサービスを提供しています。 

業務効率化やテレワーク環境の整備等早期に実現することができます。 

プラン一覧

Microsoft 365導入サービスでは、以下の通り組み合わせサービスを提供しています。お客様のご希望により、プランをカスタマイズすることも可能です。 

1.ライセンス販売プラン 

Microsoft 365のライセンスのみの販売となります。ヒアリングを行い最適なプランをご案内いたします。 

2.スタンダードプラン 

Microsoft 365の利用を開始するために必要な最低限の設定を低価格で行います。 

3.アドバンスドプラン 

Microsoft 365の初期設定及びメール、ファイル共有、Azure ADの初期設定を行います。 

4.カスタマイズプラン 

その他お客様のご要望に合わせて設定作業を行います。
